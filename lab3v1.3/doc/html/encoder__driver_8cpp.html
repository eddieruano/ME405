Detailed Description

This class is the encoder driver class.

This class creates an instance of a motor assuming caller gives all the correct and legal addresses and pin values, driver is able to map a motor to one of the two H bridge modules on the Custom ME 405 board.

ISR ( INT6_vect  )

This ISR triggers when a rising or falling edge of a wave is detected on pin E6.

This ISR checks to see if a condition is true, if both waves are at high then we can arbitrarily call that clockwise or countercw and deduce logic from there as long as we are consistent in ISR7.

Parameters
[in]INT&_vecta constant value mask that tells this ISR that the pin it need to look out for is pin E6
