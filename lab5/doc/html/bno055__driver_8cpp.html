Detailed Description

Main driver for the Adafruit BNO055 IMU Sensor. Based heavily on KTown's driver created for the Arduino line, however optimized for speed.

Based on KTown's layout, this driver focuses on obtaining data in a fast and constant way. It uses the MAX 400 KHz speed on the BNO055 and sets the IMU at Startup. The IMU is ready to go approx 5 sec after it is powered on and can update every 100 ms or so before being slow.
