\subsection{Detailed Description}
This is the file for the 'task\-\_\-encoder' class which handles the encoder\-\_\-driver class. This class is given a pointer to an 'encoder\-\_\-driver' class which then handles the initialization of all the registers. It holds the highest priority since the encoder has valuable information which we cannot miss. it runs every 10 ms, and calculates how many ticks have elapsed, giving us a per second reading.
