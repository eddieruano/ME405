\subsection{Detailed Description}
This is the file for the '\hyperlink{classtask__user}{task\-\_\-user}' class that is what any user will mainly interact with. Built on top of J\-R Ridgely's architecture, this task uses case statements that infinitely loop in order to perform certain operations. In this version of this class (v1.\-1) the only module avaliable is the 'Main Motor Module' which allows the user to control both motors in on either H\-Bridge simultaneously.
