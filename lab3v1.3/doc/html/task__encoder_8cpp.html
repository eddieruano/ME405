Detailed Description

This is the file for the 'task_encoder' class which handles the encoder_driver class.

This class is given a pointer to an 'encoder_driver' class which then handles the initialization of all the registers. It holds the highest priority since the encoder has valuable information which we cannot miss. it runs every 10 ms, and calculates how many ticks have elapsed, giving us a per second reading.
