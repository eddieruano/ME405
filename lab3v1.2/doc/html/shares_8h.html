Detailed Description

This file contains extern declarations for queues and other inter-task data communication objects used in a ME405/507/FreeRTOS project.

This is a task class that will control the operation of its own motor driver given to it as a pointer and initiated in the main() function. it lays out the logic necessary to make commands from task_user to the motor driver happen.
