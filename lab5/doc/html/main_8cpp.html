Detailed Description

Contains main function in which RTOS libraries are loaded and the task scheduler is started.

The main function contains the instance of the bno055 we're going to test an the task IMU which currently only updates the shares with the values in the bno055. If we want to change the pins for the BNO055, we unfortunately can't because the only i2c pins on the board are wired to PD0, and PD1 which in turn are IRQ an CE on the MiRf socket.
