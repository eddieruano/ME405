Detailed Description

This is the header file for the task_motor class.

It lays out prototypes for the task_motor constructor and the run class which is constantly called during operation. It also declared 3 local variables that hold a 'motor_driver' pointer, a unsigned byte holding the task identifier (1 or 2) which is checked for at the beginning of the task logic and a pointer to an 'adc' A/D converter class.
