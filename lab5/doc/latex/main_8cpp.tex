\subsection{Detailed Description}
Contains main function in which R\-T\-O\-S libraries are loaded and the task scheduler is started. The main function contains the instance of the bno055 we're going to test an the task I\-M\-U which currently only updates the shares with the values in the bno055. If we want to change the pins for the B\-N\-O055, we unfortunately can't because the only i2c pins on the board are wired to P\-D0, and P\-D1 which in turn are I\-R\-Q an C\-E on the Mi\-Rf socket.
